# For teams

Everyone clones the repo, then follows **[GETTING_STARTED.md](GETTING_STARTED.md)** on their own laptop.

- Each person keeps their own **`.env`** and **`credentials.json`** (never commit them).
- Share the **Founders-UIUC-Exploration** repo link; no need to share secrets.
- If `requirements.txt` changes, run:

```bash
.venv/bin/pip install -r requirements.txt
```

That’s it — the detailed install steps are only in **GETTING_STARTED.md** so we don’t maintain two guides.
